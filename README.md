# Disciplina de Algoritmos e Estrutura de Dados 2

Este repositório contém alguns conteúdos da disciplina de Algoritmos e Estrutura de Dados, organizados em duas seções:

## Prova

A pasta `/prova` contém os exercícios destinados às provas da disciplina.

## Lista de Exercícios

A pasta `/lista` está organizada de forma que cada questão é uma biblioteca. Cada biblioteca contém os arquivos relacionados à resolução de um exercício específico.
